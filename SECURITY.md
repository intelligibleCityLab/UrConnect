# Security Policy

## Supported Versions

UrConnect is preparing its first public release. Until a stable release exists, security fixes are handled on the `main` branch.

## Reporting A Vulnerability

Please do not open a public issue for vulnerabilities involving file parsing, native code execution, build artifacts, or private data exposure. Contact the maintainers privately through the repository owner account or an agreed institutional contact.

Include:

- affected commit or release
- operating system
- input file type
- reproduction steps
- expected impact

We will acknowledge valid reports and coordinate a fix before public disclosure.
