# Code of Conduct

UrConnect follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Standard

We expect participants to communicate respectfully, focus on the work, and make technical discussions useful for maintainers, users, and researchers.

Unacceptable behavior includes harassment, personal attacks, discriminatory language, doxxing, or publishing private data without permission.

## Enforcement

Project maintainers may remove comments, close issues, restrict participation, or take other moderation actions when needed. Serious issues should be reported privately to the maintainers before being discussed publicly.

## Reference

The full Contributor Covenant text is available at https://www.contributor-covenant.org/version/2/1/code_of_conduct/
