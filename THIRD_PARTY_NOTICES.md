UrConnect is licensed as a whole under GPLv3 because it is derived from GPLv3-or-later depthmapX/sala components. This file summarizes important bundled third-party or inherited components. It is not a substitute for the source-level copyright and license notices.

## depthmapX and sala Components

Many files under `depthmapX/`, `salalib/`, and `mgraph440/` retain copyright and GPL notices from depthmapX and sala contributors, including University College London, Alasdair Turner, and Tasos Varoudis.

The relevant source files state GPLv3-or-later terms. The repository root includes the GPLv3 text in `LICENSE`.

## genlib

Files under `genlib/` include components from the depthmapX-related genlib codebase. Some files carry GNU Lesser General Public License notices. The LGPL text is retained in `genlib/lgpl.txt`.

## Shapelib

Files under `SNDAApp/lib/` include Shapelib sources. Shapelib source headers state that the software is available under an MIT-style license, or alternatively under LGPL terms. The bundled Shapelib `COPYING` file is retained in `SNDAApp/lib/COPYING`.
