# 快速开始

## 五步开始

1. 在 GIS 中准备并清理线段化街道网络。
2. 使用 `File > Open` 打开网络。
3. 在 `Analysis` 中选择分析类型，例如 `Reach`、`Distance` 或 `Point Distance`。
4. 设置参数，然后点击 `Run`。
5. 查看地图可视化、导出的属性和生成字段。

## 数据要求

UrConnect 使用线段模型。每条线代表一个街道线段。当前程序不编辑几何图形，因此拓扑关系、端点吸附、重复线段和属性连接应在打开文件前完成。

支持的输入格式：

- Shapefile (`.shp`)
- 包含线段端点坐标的 CSV 文件
- 包含线段端点坐标的 TXT 文件

对于 Shapefile 输入，分析结果会写入源文件。对于 CSV/TXT 输入，UrConnect 也会创建同名 Shapefile，便于在 GIS 中使用结果。

## 核心概念

可达量
: 在某种距离规则下，从源线段可达的网络总量。总量可以是线段长度、交叉口数量或所选加权属性。

转向距离
: 基于转向的拓扑距离。可通过角度阈值忽略较小角度变化。

交叉口距离
: 基于指定度数交叉口数量的距离，例如三岔路口或十字路口。

加权可达量和距离
: 使用人口、POI 数量、建筑面积或其他 GIS 连接变量等数值字段进行属性加权计算。
