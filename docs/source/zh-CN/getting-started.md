# 快速开始

## 五步开始

1. 在 GIS 中准备并清理线段化街道网络。
2. 使用 `File > Open` 打开 Shapefile、CSV 或 TXT。
3. 在 `Analysis` 中选择 `Reach`、`Distance` 或 `Point Distance`。
4. 输入参数并点击 `Run`。
5. 查看地图可视化、属性字段和导出结果。

```{figure} ../_static/images/guide/4.32.png
:alt: UrConnect 软件界面

UrConnect 软件界面。
```

UrConnect 只支持线段模型，不提供图形编辑功能。拓扑清理、端点吸附、重复线检查和属性连接应在进入 UrConnect 前完成。

## 核心概念

可达量
: 在指定距离规则下，从某线段可达的网络总量。总量可以是线段长度、交叉口数量或加权属性。

转向距离
: 以转弯次数为基础的拓扑距离，可通过角度阈值忽略较小角度变化。

交叉口距离
: 以特定度数交叉口数量为基础的距离，例如三岔或十字路口。

加权分析
: 使用人口、POI、建筑面积等数值字段作为分析权重。
