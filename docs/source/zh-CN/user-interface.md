# 用户界面

UrConnect 使用基于面板的布局，而不是依赖大量菜单操作。大多数操作被组织在可折叠的单选面板中。

## 主要区域

工具与数据管理面板
: 缩放、平移、选择、打开文件、导出屏幕和导出属性。

分析面板
: 包含 Reach、Interactive Reach、Distance 和 Point Distance 模块。

路径分析面板
: 在所选起点/终点线段或 OD 表之间模拟最短路径。

数据可视化面板
: 控制背景、色谱范围、图层显示、线色和线宽。

摘要区域
: 显示文件名和临时分析摘要。

地图工作窗口
: 显示网络和计算结果可视化。

地图数据属性栏
: 显示已打开数据集中的数值字段。文本字段会被隐藏。

## 运行分析

选择一个分析模式，设置参数，然后点击左侧面板底部的 `Run`。使用 `Stop` 可取消耗时较长的计算。

## 显示属性

使用 `Show Attributes` 打开或关闭右侧属性栏。改变色谱范围前，先选择一个数值字段作为当前可视化字段。
