# 应用案例

UrConnect 可以把几何-拓扑网络结构与街道属性和 GIS 变量结合起来。这使得指标更容易跨案例比较，也适用于城市设计、规划和城市形态研究。

## 跨城市转向可达量比较

```{figure} ../_static/images/guide/4.59.png
:alt: 杭州和上海转向可达量比较

杭州和上海两次方向变化可达量比较。
```

指南示例比较了杭州主城区和上海内环区域。尽管两者的街道密度和平均线段长度相近，全局两次转向可达量差异明显，说明它们在城市尺度结构角色上存在差异。

## 结合人口和 POI 数据的加权可达量

```{figure} ../_static/images/guide/4.60.png
:alt: 上海加权可达量

上海街道长度、人口和 POI 可达量。
```

加权可达量可以展示功能中心性与街道网络中心性的差异。在上海示例中，POI 可达量比单纯街道长度可达量更突出外滩区域。
