# 應用案例

UrConnect 可以把幾何-拓撲網路結構與街道屬性和 GIS 變量結合起來。這使得指標更容易跨案例比較，也適用於城市設計、規劃和城市形態研究。

## 跨城市轉向可達量比較

```{figure} ../_static/images/guide/4.59.png
:alt: 杭州和上海轉向可達量比較

杭州和上海兩次方向變化可達量比較。
```

指南示例比較了杭州主城區和上海內環區域。儘管兩者的街道密度和平均線段長度相近，全域兩次轉向可達量差異明顯，說明它們在城市尺度結構角色上存在差異。

## 結合人口和 POI 資料的加權可達量

```{figure} ../_static/images/guide/4.60.png
:alt: 上海加權可達量

上海街道長度、人口和 POI 可達量。
```

加權可達量可以展示功能中心性與街道網路中心性的差異。在上海示例中，POI 可達量比單純街道長度可達量更突出外灘區域。
