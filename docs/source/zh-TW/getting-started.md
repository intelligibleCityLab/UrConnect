# 快速開始

## 五步開始

1. 在 GIS 中準備並清理線段化街道網路。
2. 使用 `File > Open` 開啟網路。
3. 在 `Analysis` 中選擇分析類型，例如 `Reach`、`Distance` 或 `Point Distance`。
4. 設定參數，然後點擊 `Run`。
5. 查看地圖視覺化、匯出的屬性和生成欄位。

## 資料要求

UrConnect 使用線段模型。每條線代表一個街道線段。當前程式不編輯幾何圖形，因此拓撲關係、端點吸附、重複線段和屬性連接應在開啟文件前完成。

支援的輸入格式：

- Shapefile (`.shp`)
- 包含線段端點座標的 CSV 文件
- 包含線段端點座標的 TXT 文件

對於 Shapefile 輸入，分析結果會寫入源文件。對於 CSV/TXT 輸入，UrConnect 也會建立同名 Shapefile，便於在 GIS 中使用結果。

## 核心概念

可達量
: 在某種距離規則下，從源線段可達的網路總量。總量可以是線段長度、交叉口數量或所選加權屬性。

轉向距離
: 基於轉向的拓撲距離。可透過角度閾值忽略較小角度變化。

交叉口距離
: 基於指定度數交叉口數量的距離，例如三岔路口或十字路口。

加權可達量和距離
: 使用人口、POI 數量、建築面積或其他 GIS 連接變量等數值欄位進行屬性加權計算。
