# 快速開始

## 五步開始

1. 在 GIS 中準備並清理線段化街道網路。
2. 使用 `File > Open` 開啟 Shapefile、CSV 或 TXT。
3. 在 `Analysis` 中選擇 `Reach`、`Distance` 或 `Point Distance`。
4. 輸入參數並點擊 `Run`。
5. 查看地圖視覺化、屬性欄位和匯出結果。

:alt: UrConnect 軟體介面

UrConnect 軟體介面。

UrConnect 只支援線段模型，不提供圖形編輯功能。拓撲清理、端點吸附、重複線檢查和屬性連接應在進入 UrConnect 前完成。
