# 使用者介面

UrConnect 使用基於面板的佈局，而不是依賴大量選單操作。大多數操作被組織在可折疊的單選面板中。

## 主要區域

工具與資料管理面板
: 縮放、平移、選擇、開啟文件、匯出畫面和匯出屬性。

分析面板
: 包含 Reach、Interactive Reach、Distance 和 Point Distance 模組。

路徑分析面板
: 在所選起點/終點線段或 OD 表之間模擬最短路徑。

資料視覺化面板
: 控制背景、色譜範圍、圖層顯示、線色和線寬。

摘要區域
: 顯示文件名和臨時分析摘要。

地圖工作視窗
: 顯示網路和計算結果視覺化。

地圖資料屬性欄
: 顯示已開啟資料集中的數值欄位。文字欄位會被隱藏。

## 執行分析

選擇一個分析模式，設定參數，然後點擊左側面板底部的 `Run`。使用 `Stop` 可取消耗時較長的計算。

## 顯示屬性

使用 `Show Attributes` 開啟或關閉右側屬性欄。改變色譜範圍前，先選擇一個數值欄位作為當前視覺化欄位。
