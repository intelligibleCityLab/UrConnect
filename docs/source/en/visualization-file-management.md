# Visualization And File Management

## Visualization

```{figure} ../_static/images/guide/4.56.png
:alt: Visualization panel

Visualization panel.
```

The Visualization module controls background color, value color ranges, layer visibility, line color, and line width.

Color Range
: Changes the color ramp for the active numeric attribute.

Layers
: Toggles `Reach by Selection` and `Path Analysis` output layers.

Lines Edit
: Applies line color or width settings. Line width can be filtered by a selected numeric attribute.

```{figure} ../_static/images/guide/4.57.png
:alt: Color range settings

```

```{figure} ../_static/images/guide/4.58.png
:alt: Line width settings

Line width settings.
```

Use `Reset` to restore the default visualization state.

## File Management

Open
: Opens Shapefile, CSV, or TXT segment-network data.

Export Analysis
: Exports `Interactive Reach` or `Path Analysis` generated layers as Shapefile.

Export Screen
: Exports the current view as SVG or PNG.

Export Attributes
: Exports selected numeric attributes and segment IDs to TXT or CSV.

For Shapefiles, fields are limited to 10 characters. CSV and TXT outputs use longer field names when possible.
